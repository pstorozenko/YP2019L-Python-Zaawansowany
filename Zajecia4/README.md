Dziś zajmiemy się Flaskiem :)

Jest to popularny mikro-framework służący do robienia małych appek/stron/serwisów webowych.

Konsolę otwieramy przez `ctrl + ` `

I tam wpisujemy:

```
set FLASK_APP=[nazawa pliku]
set FLASK_ENV=development
flask run
```
